## Run Locally

Clone the project

```bash
  git clone https://github.com/IMohy/nextjs-portfolio.git

```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  pnpm i
```

Start the server

```bash
  pnpm dev
```
